### Development & Engineering
- **algorithmic-art** - Creating algorithmic art using p5.js with seeded randomness and interactive parameter exploration
- **canvas-design** - Create beautiful visual art in .png and .pdf documents using design philosophy
- **command-development** - Guide for creating slash commands for Claude Code
- **coding** - General coding and development capabilities
- **frontend-design** - Create distinctive, production-grade frontend interfaces
- **githubskill** - Comprehensive GitHub repository management and git operations
- **mcp-builder** - Guide for creating high-quality MCP (Model Context Protocol) servers
- **mcp-integration** - Integrate Model Context Protocol servers into Claude Code plugins
- **skill-creator** - Guide for creating effective skills that extend AI capabilities
- **webapp-testing** - Toolkit for interacting with and testing local web applications using Playwright

### Documentation & Writing
- **doc-coauthoring** - Structured workflow for co-authoring documentation
- **docs-write** - Write documentation following Metabase's conversational style
- **docx** - Comprehensive document creation and editing with .docx files
- **internal-comms** - Resources for writing internal communications
- **link-explainer** - Transform complex web articles into simple, animated HTML explanations
- **pdf** - Comprehensive PDF manipulation toolkit
- **pptx** - Presentation creation, editing, and analysis
- **xlsx** - Comprehensive spreadsheet creation, editing, and analysis

### Design & Creative
- **brand-guidelines** - Apply Anthropic's official brand colors and typography
- **brainstorming** - Explore user intent, requirements and design before implementation
- **planning-with-files** - File-based planning and documentation system
- **slack-gif-creator** - Create animated GIFs optimized for Slack
- **theme-factory** - Toolkit for styling artifacts with pre-set themes
- **ui-ux-pro-max** - Professional UI/UX across multiple platforms

### AI & Automation
- **notebooklm-py** - Unofficial Google NotebookLM Python API
- **paper-to-webpage** - Transform academic papers into web content
- **playwright** - Browser automation via Playwright MCP
- **prompt-lookup** - Discover, retrieve, and improve AI prompts
- **skill-lookup** - Find and install reusable AI capabilities
- **superpowers** - Complete software development workflow for coding agents

2. Test changes locally
3. Ensure compatibility with other skills
4. Document breaking changes
